# GitInsight – GitHub Portfolio Analyzer

## Description
GitInsight is a web application that analyzes a developer’s GitHub profile and generates a readiness score with improvement suggestions using the GitHub API.

## Features
- Repository analysis
- Score generation
- Suggestions for improvement
- Language detection
- Activity check
- README detection

## Tech Stack
Python, Flask, HTML, CSS, GitHub REST API

## How to Run
1. Install Python
2. pip install flask requests
3. python app.py
4. Open http://127.0.0.1:5000
